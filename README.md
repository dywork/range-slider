# RangeSlider (jQuery plugin)

[Просмотр](https://dywork.github.io/range-slider/)

## Установка

```
$ npm install
```

## Команды

**Клонировать репозиторий: **
`git clone https://github.com/dywork/range-slider`

**Запуск: **
`npm start`

**Сборка: **
`npm run build`

**Тесты: **
`npm run test`

**Деплой на github page: **
`npm run deploy`
